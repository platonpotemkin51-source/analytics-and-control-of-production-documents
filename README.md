# WoodFlow — веб-приложение для деревообрабатывающей логистики

Приложение на Python/Flask для учета пачек древесины, формирования отправок машин, аналитики прибыли и архива.

## Возможности
- Главная страница: ключевые KPI + графики (Chart.js).
- Страница склада: добавление и просмотр пачек.
- Страница отправки машин: выбор пачек чекбоксами и формирование отправки.
- Архив: история отправок и привязанных пачек.

## Запуск
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Откройте: `http://localhost:5000`
